# BookYol Affiliate Engine

WordPress plugin for [BookYol.com](https://bookyol.com) — book affiliate link management with geo-routing, click tracking, auto book lookup, and display shortcodes.

## Features

- 📚 Book custom post type with full metadata
- 🌍 Geo-routing (US/UK/EU/GCC) — shows the right affiliate link per visitor location
- 🔗 Auto-generate affiliate links via ISBN (Everand, Libro.fm, Ebooks.com, Bookshop.org, Kobo, Jamalon)
- 📊 Click tracking with admin dashboard widget
- 🔍 Google Books API lookup — auto-fill book metadata
- 📦 Bulk import — add 20+ books at once
- 🏠 Custom homepage template (Astra-compatible)
- ⚙️ Full homepage settings with 10 configurable sections

## Plugin Files

All plugin files are inside the `bookyol-affiliate-engine/` directory. Upload this folder as a WordPress plugin.
